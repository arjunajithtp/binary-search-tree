# binary-search-tree
## binary-search-tree
Binary search tree
##### Usage
```
t := bst.BST{}
```
### Methods
#### Insert()
Insert method inserts data into the list.
##### Usage
```
t.Insert(5) 

or

t.Insert(5, 6, 7, 8)
```
#### CollectData()
This method collects data from the BST according to the tree traversal order passed to it.
##### Usage
```
t.CollectData(bst.Preorder)
```
